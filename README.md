# Student Management System

A Django-based web application to manage student records.

## Features
- Add, view, edit, and delete students
- Django Admin panel integration
- Responsive UI using Bootstrap

## Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, Bootstrap

## How to Run the Project

1. Clone the repository
   ```bash
   git clone https://github.com/Singireddynithin/student-management-django/tree/main/student_project
2. Navigate to the project folder
   cd student-management-django
3. Create and activate virtual environment
   python -m venv venv
   source venv/Scripts/activate
4. Install dependencies
   pip install django
5. Run database migrations
   python manage.py migrate
6. Run the server
   python manage.py runserver
7. Open in browser (after running the server locally):
http://127.0.0.1:8000/
8. Admin Panel:
http://127.0.0.1:8000/admin/
